# Sampahin.co
SAMPAHIN adalah sebuah konsep inovasi yang dikembangkan sebagai aplikasi digital untuk pengelolaan sampah.  Tujuan utamanya adalah untuk mengelola sampah secara lebih efisien menggunakan teknologi digital, sekaligus menumbuhkan jiwa sosiopreneurship (kewirausahaan sosial) dan mendukung pengembangan ekonomi berbasis digital di kalangan penggunanya.
